# Admin Panel

This is the Admin Dashboard for the Tomato Food Delivery Application. It is built using **React** and **Vite** and connects to the backend API to manage food items and track customer orders.

## 🛠️ Features
- **Add Dishes**: Create new food items by uploading images, selecting categories, and setting prices.
- **List & Remove Dishes**: View all existing items in the database and delete them when out of stock.
- **Manage Orders**: Track real-time customer orders, view billing/address details, and update the status (`Food Processing` ➔ `Out for delivery` ➔ `Delivered`).

## 🚀 Quick Start
1. Ensure the backend server is running.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Access the panel at `http://localhost:5174` (or port specified in terminal).
