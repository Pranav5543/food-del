# food -del - Customer Frontend

This is the customer-facing interface for the Tomato Food Delivery Application. It is built using **React**, **Vite**, and custom responsive CSS.

## 🛠️ Features
- **Dynamic Menu Exploration**: Browse categories and filter dishes instantly.
- **Cart Management**: Add/remove items with state-persisted count and total calculation.
- **User Authentication**: Secure Login/Sign-up modals with JWT sessions.
- **Stripe Checkout**: Integrated checkout page redirecting to secure Stripe payments.
- **Order Tracking**: Track order status and payment verification history.

## 🚀 Quick Start
1. Ensure the backend server is running.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Access the application at `http://localhost:5173`.
